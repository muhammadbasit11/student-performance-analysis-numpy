Project Overview:
This project simulates 200 students’ scores in Physics, Biology, English, Statistics, and Maths. Key analyses include calculating averages, identifying top students, and finding subjects with the highest performance.

Key Features & Analysis:
Average marks per student across all subjects.
Count of students scoring >85 in Maths.
Top 10 students by average marks.
Subject with the highest average marks.


Skills & Concepts Applied:
Numpy array operations (mean, sum, argsort, argmax).
Data aggregation and indexing.
Ranking and selection of top-performing students.
